常见问题

如何添加新的食材？

1. 点击左上角 +

2. 选择食材类别

3. 选择具体食材并设置存储位置

4. 点击确认添加

如何设置提醒时间？

1. 进入"模版调整"页面

2. 点击需要修改的食材

3. 调整保质期和提醒时间

4. 保存更改

为什么没有收到通知？

请检查以下设置：

  • 确保在系统设置中允许应用发送通知
  • 检查应用内是否正确设置了提醒时间
  • 确保没有开启"勿扰模式"

FAQ

How to add new food items?

1. Click the "+" button in the left top

2. Select food category

3. Choose specific food item and set storage location

4. Confirm to add

How to set reminder time?

1. Go to "Template Settings" page

2. Click on the food item you want to modify

3. Adjust expiration and reminder time

4. Save changes

Why am I not receiving notifications?

Please check the following settings:

  • Ensure notifications are allowed in system settings
  • Check if reminder time is correctly set in the app
  • Make sure "Do Not Disturb" mode is off
